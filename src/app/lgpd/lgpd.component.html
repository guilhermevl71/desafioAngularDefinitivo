LGPD - Lei Geral de Proteção de Dados

Esta página tem como objetivo informar os usuários sobre como os seus dados pessoais são tratados em conformidade com a Lei Geral de Proteção de Dados Pessoais (Lei nº 13.709/2018).

1. O que é a LGPD?

A LGPD é a legislação brasileira que regulamenta o tratamento de dados pessoais, garantindo mais segurança, privacidade e controle para os cidadãos sobre suas informações.

2. Quais dados coletamos?

Podemos coletar dados como nome, e-mail, telefone, localização e preferências de veículos, a partir de formulários, navegação no site ou cadastro.

3. Finalidade do uso dos dados

Os dados são utilizados para personalizar sua experiência, entrar em contato com você, oferecer promoções exclusivas e melhorar nossos serviços.

4. Seus direitos segundo a LGPD

  • Acessar e confirmar a existência de tratamento de dados;
  • Corrigir dados incompletos, inexatos ou desatualizados;
  • Solicitar a exclusão de dados tratados com consentimento;
  • Revogar o consentimento a qualquer momento.

5. Segurança das informações

Adotamos medidas técnicas e administrativas para proteger seus dados contra acessos não autorizados, vazamentos ou alterações indevidas.

6. Encarregado pelo tratamento

Para exercer seus direitos ou tirar dúvidas sobre a LGPD, entre em contato com nosso Encarregado de Proteção de Dados por meio da página de Contato.
